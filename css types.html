3ways of CSS styling

CSS Applying Types

Hello world...CSS can be applied in 3ways
-> Inline
-> Inernal
-> External

hello Iam { Inline Styling }
Welcome to Inline styling
Inline styling can be applied using "style" attribute to an element.
Inline can apply preferably in "body" content

Hello Iam { Internal styling }
Internal styling can be applied within "style" Element. attribute Internal styling must be applied only in "head" section

Hello Iam { External Styling }
External styling can be applied by using seperate file and must save filename with ".css" extension
And add this stylesheet with "link" tag in the "head" section of html file.
